# controle-financas

Repositório de documentação e especificações para um produto de **finanças pessoais**: registro de receitas e despesas, categorias, resumo mensal e metas simples. Público-alvo: pessoas que querem mais controle de gastos e base para poupar — **CLT**, **autônomos**, **estudantes** e perfis mistos.
